# SpecLedger Homebrew Tap

This is the official [Homebrew](https://brew.sh/) tap for [SpecLedger](https://specledger.io).

## Installation

```bash
# Add the tap
brew tap specledger/homebrew-specledger

# Install SpecLedger
brew install specledger
```

## Usage

After installation, you can use SpecLedger:

```bash
# Create a new project
sl new

# Check installation
sl doctor

# Manage dependencies
sl deps list
```

## Updating

```bash
# Update SpecLedger
brew upgrade specledger

# Or update everything
brew update && brew upgrade
```

## Uninstallation

```bash
# Uninstall SpecLedger
brew uninstall specledger

# Remove the tap
brew untap specledger/homebrew-specledger
```
